# 實作簡單express後端

**請於啟動前確保設置環境變數正確** \
資料庫[mongodb atlas](https://www.mongodb.com/) \
檔案存儲在[google cloud](https://cloud.google.com/)需要申請金鑰並改名為**mykey.json**
____

### `npm start`

以開發模式啟動 \
預設[http://localhost:5000](http://localhost:5000)開啟

### `npm run build`

將後端打包並且優化

#### 啟動
````
npm run prod
````
